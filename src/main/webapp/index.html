Use the following to request a new unique token: Request Token

The base url for storing and saving is the following: Data base url

For saving an entry, use PUT method an adding the following path parameters: "rest/data/{token}/{key}/{value}"

For retrieving an entry, use GET method an adding the following path parameters: "rest/data/{token}/{key}"

For retrieving all entries for a given token, use GET method an adding the following path parameters: "rest/data/{token}"
